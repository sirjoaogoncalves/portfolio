SimpleSDK

github.com/sirjoaogoncalves/SimpleSDK

C++ library providing TCP client/server functionality and SQLite database integration for RFID tag information processing. Designed to read, output, and store RFID tag data when scanned under antenna systems.

Features real-time RFID data capture with tag ID, antenna identification, timestamp recording, and RSSI (Received Signal Strength Indicator) measurements. Includes comprehensive database storage for inventory tracking and tag history management.

Built with C++ using Boost.Asio for asynchronous networking, SQLite3 for embedded database storage, and custom circular buffer implementation for efficient data processing. Cross-platform support for Linux, macOS, and Windows.

Architecture: Modular design with separate TCP client/server components, database abstraction layer, and circular buffer for data streaming. Supports multiple simultaneous antenna connections and concurrent RFID tag processing.

Usage: ./simple_sdk_example - Starts TCP server on configurable port, processes incoming RFID data packets, and stores readings in SQLite database with automatic tag inventory management.

HomeNetHelper

github.com/sirjoaogoncalves/homenethelper

Network monitoring tool written in Go that provides real-time information about devices connected to your home network.

Captures and analyzes network traffic to give insights into device usage, connection types, and data transfer rates. Monitors multiple interfaces simultaneously (Ethernet and Wi-Fi) and identifies top protocols and ports used by each device.

Built with github.com/google/gopacket for packet capture and libpcap for low-level network access. Uses concurrent packet processing with goroutines and terminal UI with real-time statistics updates.

Usage: sudo ./homenethelper -interfaces=eth0,wlan0 -refresh=5

Outputs table with device IP, hostname, connection type, interface, transfer speeds, and protocol/port usage statistics. Requires sudo privileges for packet capture.

Trading Terminal

github.com/sirjoaogoncalves/trading_terminal

Terminal-based cryptocurrency trading dashboard built with OCaml.

Features real-time cryptocurrency price monitoring, interactive charts with multiple timeframes (1H/4H/1D/1W), market overview with multiple pairs, and keyboard-based navigation. Clean TUI designed for efficiency.

Built with lwt for asynchronous I/O, notty for terminal graphics, and cohttp-lwt-unix for HTTP requests. Integrates multiple cryptocurrency APIs (Binance, KuCoin, CoinGecko, CryptoCompare) with automatic failover.

Usage: dune exec trading_terminal

Navigation: Tab to switch views, arrow keys for selection, number keys (1/4/d/w) for timeframes, 'r' to refresh, 'q' to quit. Supports major cryptocurrencies with real-time updates.
